hydrosdk.cluster module¶

class hydrosdk.cluster.Cluster(http_address: str, grpc_address: Optional[str] = None, ssl: bool = False, grpc_credentials: Optional[grpc.ChannelCredentials] = None, grpc_options: Optional[Dict] = None, grpc_compression: Optional[grpc.Compression] = None)¶

Bases: object

Cluster responsible for server interactions

Creates a Cluster object which hides networking details and provides a connection to a deployed Hydrosphere cluster.

Parameters

  • http_address (str) – HTTP connection address of a Hydrosphere cluster

  • grpc_address (Optional[str]) – gRPC connection address of a Hydrosphere cluster

  • ssl (bool) – Whether to use an SSL-enabled channel for a gRPC connection

  • grpc_credentials (Optional[grpc.ChannelCredentials]) – An optional ChannelCredentials instance

  • grpc_options (Optional[Dict]) – An optional list of key-value pairs (channel args in gRPC Core runtime) to configure the channel

  • grpc_compression (Optional[grpc.Compression]) – An optional value indicating the compression method to be used over the lifetime of the channel

Examples

Example of creating a Cluster:

# Cluster with only an HTTP connection
Cluster("your-cluster-url")

# Example of creating a Cluster with an HTTP and a secured gRPC connection::
from grpc import ssl_channel_credentials
Cluster("http-cluster-address", grpc_address="grpc-cluster-address", ssl=True, grpc_credentials=ssl_channel_credentials())
